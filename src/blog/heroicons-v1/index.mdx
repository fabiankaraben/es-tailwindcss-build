## Librerías React + Vue

Además de obtener los iconos que necesitas directamente desde el sitio web, ahora puedes instalar nuestras librerías oficiales para React y Vue para un acceso rápido y fácil a cada icono como un componente dedicado.

Así es como se ve con React, por ejemplo:

```jsx
import { BeakerIcon } from "@heroicons/react/solid";

function MyComponent() {
  return (
    <div>
      <BeakerIcon className="h-5 w-5 text-blue-500" />
      <p>...</p>
    </div>
  );
}
```

[Consulta la documentación](https://github.com/tailwindlabs/heroicons) en GitHub para saber más.
