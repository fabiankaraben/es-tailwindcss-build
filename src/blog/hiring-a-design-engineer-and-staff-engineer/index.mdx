## Ingeniero de Diseño

Estamos contratando un [Ingeniero de Diseño](https://v3.tailwindcss.com/careers/design-engineer) para construir interfaces ambiciosas, prototipar nuevas ideas y empujar los límites de lo que es posible con Tailwind CSS.

<Link href="https://v3.tailwindcss.com/careers/design-engineer">
  <Image src={designEngineer} alt="Ingeniero de diseño" />
</Link>

Serías responsable de cosas como:

- **Diseñar y construir sitios web de marketing ambiciosos** para nuestros proyectos de código abierto, productos comerciales y eventos como Tailwind Connect.
- **Diseñar y prototipar nuevas características para Tailwind CSS** para asegurarnos de que siempre estemos usando todo el potencial de la plataforma.
- **Crear nuevos componentes y plantillas para Tailwind UI**, llevándolos desde el concepto inicial hasta el lanzamiento.
- **Mejorar nuestra documentation con demostraciones visuales** para facilitar que las personas comprendan y apliquen características complejas de CSS en su trabajo.
- **Enseñar e inspirar a nuestra audiencia** desglosando cosas interesantes que diseñes y construyas como artículos y publicaciones en redes sociales.

Aquí tienes algunos ejemplos reales de proyectos en los que habrías trabajado en los últimos meses:

- **Diseñar y construir el micrositio [Tailwind Connect](https://connect.tailwindcss.com/)** — incluyendo la creación de una experiencia única de diseño de insignias impulsada por Markdown.
- **Crear los [primeros componentes](https://catalyst.tailwindui.com/docs/dialog) para Catalyst** — nuestro primer kit de UI de React totalmente componentizado.
- **Construir una herramienta para capturar videos para nuestro sitio [Showcase](https://tailwindcss.com/showcase)** — para automatizar la creación de videos de vista previa que se sientan como una persona real desplazándose por un sitio.
- **Construir [demostraciones interactivas](https://tailwindcss.com/docs/height#dynamic-viewport-height) para explicar las unidades dinámicas de viewport** — para incluir en la documentación junto con el lanzamiento de Tailwind CSS v3.4.

Después de que empieces, trabajarías en próximos proyectos como:

- **Diseñar y construir un micrositio interactivo para el lanzamiento de Tailwind CSS v4.0** con demostraciones cuidadosamente elaboradas que comuniquen las mejoras más importantes de manera visual.
- **Prototipar APIs para animaciones controladas por scroll en Tailwind CSS**, encontrando el equilibrio adecuado entre simplicidad y flexibilidad, y haciendo que nuestros usuarios sientan que tienen superpoderes.
- **Explorar nuevas paletas de colores con soporte automático para modo oscuro**, determinando cuidadosamente las abstracciones correctas para diferentes niveles de jerarquía que se apliquen universalmente a diferentes proyectos.
- **Investigar y diseñar soporte para `text-shadow` en Tailwind CSS**, finalmente.

Este es un puesto totalmente remoto con un salario de **$275,000 USD**, abierto a candidatos en las zonas horarias del Este (UTC-5) a Europa Central (UTC+1).

## Ingeniero de Software de Staff

Estamos contratando un [Ingeniero de Software de Staff](https://v3.tailwindcss.com/careers/staff-software-engineer) para trabajar en Tailwind CSS, Headless UI y nuestro ecosistema de herramientas de soporte como la extensión Tailwind CSS IntelliSense para VS Code.

<Link href="https://v3.tailwindcss.com/careers/staff-software-engineer">
  <Image src={staffEngineer} alt="Ingeniero de staff" />
</Link>

Tendrías una amplia gama de responsabilidades, incluyendo:

- **Construir y documentar nuevas características** para proyectos como Tailwind CSS y Headless UI.
- **Contribuir con mejoras a proyectos upstream** como Lightning CSS, Next.js y Vite.
- **Mantener flujos de trabajo de CI y automatizaciones** para ejecutar pruebas y construir lanzamientos.
- **Gestionar issues y contribuciones** de la comunidad a nuestros repositorios de código abierto.
- **Definir y mantener estándares de ingeniería** con un enfoque en la calidad del código, el rendimiento y la documentación, para asegurarnos de que siempre entreguemos un trabajo del que podamos estar orgullosos.
- **Gestionar las prioridades del proyecto y ajustar el alcance** para cumplir con los plazos del proyecto sin sacrificar los objetivos de alto nivel.

Aquí tienes algunos ejemplos reales de proyectos en los que habrías trabajado en los últimos meses:

- **Crear una [librería de Rust](https://github.com/tailwindlabs/tailwindcss/blob/0ee3508179966a767b7802a7262bb9d4ce2221da/oxide/crates/core/src/lib.rs) para extraer nombres de clases de utilidad** — una herramienta que puede escanear un directorio completo de archivos de plantilla lo más rápido posible y extraer nombres de clases potenciales en paralelo.
- **Escribir un [parser rápido de CSS](https://github.com/tailwindlabs/tailwindcss/blob/0ee3508179966a767b7802a7262bb9d4ce2221da/packages/tailwindcss/src/css-parser.ts) en TypeScript** — para parsear solo la cantidad exacta de detalle necesaria para los componentes internos de Tailwind, buscando atajos específicos del dominio para hacerlo significativamente más rápido que las librerías existentes para nuestras necesidades.
- **Arreglar un problema con el [bloqueo de scroll](https://github.com/tailwindlabs/headlessui/pull/2891) en Headless UI** — para asegurarse de que los elementos desplazables en los diálogos todavía se puedan desplazar en iOS.
- **Hacer campaña por [nuevas APIs](https://github.com/vitejs/vite/pull/16135) que necesitamos en Vite** — preparar un argumento convincente para las mejoras que necesitamos para soportar Tailwind CSS y colaborar con el equipo central de Vite para prototipar una solución.

Después de que empieces, trabajarías en próximos proyectos como:

- **Desarrollar un plugin de bundler de Tailwind CSS para Next.js** — para generar tu CSS utilizando el grafo de módulos en lugar de escanear el sistema de archivos, trabajando con nuestros contactos en Vercel para hacerlo bien.
- **Diseñar un conjunto de APIs de integración de IDE para Tailwind CSS** — APIs de primera clase, estables y documentadas en las que los equipos de editores como JetBrains puedan confiar para agregar cosas como completados, advertencias de lint, ordenamiento de clases y más a sus herramientas.
- **Diseñar, construir y documentar un componente Tooltip para Headless UI** — asegurándose de que sea totalmente accesible, con una API flexible pero encantadora.
- **Construir una capa de compatibilidad hacia atrás para Tailwind CSS v4.0** — reintroducir el soporte para archivos de configuración de JavaScript, explorar herramientas de codemod y asegurarse de que los plugins comunitarios existentes sean lo más compatibles posible con el nuevo motor.

Este es un puesto totalmente remoto con un salario de **$275,000 USD**, abierto a candidatos en las zonas horarias del Este (UTC-5) a Europa Central (UTC+1).
