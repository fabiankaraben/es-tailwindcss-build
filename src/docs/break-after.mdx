## Ejemplos

### Ejemplo básico

Usa utilidades como `break-after-column` y `break-after-page` para controlar cómo un salto de columna o página debe comportarse después de un elemento:

```html
<!-- [!code classes:break-after-column] -->
<div class="columns-2">
  <p>Bueno, déjame decirte algo, ...</p>
  <p class="break-after-column">Claro, adelante, ríete...</p>
  <p>Tal vez podamos vivir sin...</p>
  <p>Mira. Si crees que esto es...</p>
</div>
```
