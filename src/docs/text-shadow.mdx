## Ejemplos

### Ejemplo básico

Usa utilidades como `text-shadow-sm` y `shadow-lg` para aplicar diferentes tamaños de sombras de texto a un elemento de texto:

<Figure>

<Example padding={false}>
  {
    <div className="relative grid items-center justify-around gap-4 bg-linear-45 from-indigo-400 via-purple-500 to-pink-500 px-6 py-14 text-center text-2xl font-semibold text-white">
      <p className="relative text-shadow-2xs">El rápido zorro marrón salta sobre el perro perezoso.</p>
      <p className="relative text-shadow-xs">El rápido zorro marrón salta sobre el perro perezoso.</p>
      <p className="relative text-shadow-sm">El rápido zorro marrón salta sobre el perro perezoso.</p>
      <p className="relative text-shadow-md">El rápido zorro marrón salta sobre el perro perezoso.</p>
      <p className="relative text-shadow-lg">El rápido zorro marrón salta sobre el perro perezoso.</p>
    </div>
  }
</Example>

```html
<!-- [!code classes:text-shadow-2xs,text-shadow-xs,text-shadow-sm,text-shadow-md,text-shadow-lg,text-shadow-xl] -->
<p class="text-shadow-2xs ...">El rápido zorro marrón...</p>
<p class="text-shadow-xs ...">El rápido zorro marrón...</p>
<p class="text-shadow-sm ...">El rápido zorro marrón...</p>
<p class="text-shadow-md ...">El rápido zorro marrón...</p>
<p class="text-shadow-lg ...">El rápido zorro marrón...</p>
```

</Figure>

### Cambiando la opacidad

Usa el modificador de opacidad para ajustar la opacidad de la sombra del texto:

<Figure>

<Example padding={false}>
  {
    <div className="relative grid items-center justify-around gap-4 bg-linear-45 from-indigo-400 via-sky-400 to-emerald-400 px-6 py-14 text-center text-2xl font-semibold text-white">
      <p className="relative text-shadow-lg">El rápido zorro marrón salta sobre el perro perezoso.</p>
      <p className="relative text-shadow-lg/20">El rápido zorro marrón salta sobre el perro perezoso.</p>
      <p className="relative text-shadow-lg/30">El rápido zorro marrón salta sobre el perro perezoso.</p>
    </div>
  }
</Example>

```html
<!-- [!code classes:text-shadow-lg/20,text-shadow-lg/30] -->
<p class="text-shadow-lg ...">El rápido zorro marrón...</p>
<p class="text-shadow-lg/20 ...">El rápido zorro marrón...</p>
<p class="text-shadow-lg/30 ...">El rápido zorro marrón...</p>
```

</Figure>

Las opacidades predeterminadas de las sombras de texto son bastante bajas (20% o menos), por lo que aumentar la opacidad (hasta un 50%) hará que las sombras de texto sean más pronunciadas.

### Estableciendo el color de la sombra

Usa utilidades como `text-shadow-indigo-500` y `text-shadow-cyan-500/50` para cambiar el color de una sombra de texto:

<Figure>

<Example>
  {
    <div className="relative grid h-48 place-items-center">
      <div className="-mr-10 flex gap-4 max-sm:-mr-32">
        <button className="relative rounded-full bg-linear-to-b from-sky-300 to-sky-400 to-70% px-4 py-2 text-sm font-semibold text-sky-950 shadow-md ring inset-shadow-2xs ring-sky-500 inset-shadow-white/20 text-shadow-2xs text-shadow-sky-300 dark:ring-sky-500/50">
          Reservar una demo
          <div className="absolute top-1/2 -left-6/5 size-42 -translate-y-1/2 overflow-hidden rounded-full bg-white/20 p-2 shadow-lg ring-2 ring-black/5 backdrop-blur-sm dark:ring-black/10">
            <div className="grid size-full items-center overflow-hidden rounded-full bg-linear-to-b from-sky-300 to-sky-400 to-50% px-5 pb-2 text-[70px] font-semibold whitespace-nowrap text-sky-900 inset-ring-2 inset-ring-black/10 text-shadow-[0px_5px_0px_var(--tw-text-shadow-color)] text-shadow-sky-300">
              Reservar una demo
            </div>
          </div>
        </button>
        <button className="relative rounded-full bg-linear-to-b from-white/10 to-white/20 to-70% px-4 py-2 text-sm font-semibold text-gray-950 shadow-md ring inset-shadow-2xs ring-black/20 inset-shadow-white/10 dark:text-white dark:text-shadow-2xs">
          Ver precios
        </button>
      </div>
    </div>
  }
</Example>

```html
<!-- [!code classes:text-shadow-sky-300] -->
<button class="text-sky-950 text-shadow-2xs text-shadow-sky-300 ...">Reservar una demo</button>
<button class="text-gray-950 dark:text-white dark:text-shadow-2xs ...">Ver precios</button>
```

</Figure>

Por defecto, las sombras de colores tienen una opacidad del 100%, pero puedes ajustar esto usando el modificador de opacidad.

### Eliminando una sombra de texto

Usa la utilidad `text-shadow-none` para eliminar una sombra de texto existente de un elemento:

<Figure>

```html
<!-- [!code classes:dark:text-shadow-none] -->
<p class="text-shadow-lg dark:text-shadow-none">
  <!-- ... -->
</p>
```

</Figure>
