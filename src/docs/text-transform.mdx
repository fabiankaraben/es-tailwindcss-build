## Ejemplos

### Poner texto en mayúsculas

Usa la utilidad `uppercase` para poner en mayúsculas el texto de un elemento:

<Figure>

<Example>
  {
    <p className="text-center text-lg font-medium text-gray-900 uppercase dark:text-gray-200">
      El rápido zorro marrón salta sobre el perro perezoso.
    </p>
  }
</Example>

```html
<!-- [!code classes:uppercase] -->
<p class="uppercase">El rápido zorro marrón ...</p>
```

</Figure>

### Poner texto en minúsculas

Usa la utilidad `lowercase` para poner en minúsculas el texto de un elemento:

<Figure>

<Example>
  {
    <p className="text-center text-lg font-medium text-gray-900 lowercase dark:text-gray-200">
      El rápido zorro marrón salta sobre el perro perezoso.
    </p>
  }
</Example>

```html
<!-- [!code classes:lowercase] -->
<p class="lowercase">El rápido zorro marrón ...</p>
```

</Figure>

### Capitalizar texto

Usa la utilidad `capitalize` para capitalizar el texto de un elemento:

<Figure>

<Example>
  {
    <p className="text-center text-lg font-medium text-gray-900 capitalize dark:text-gray-200">
      El rápido zorro marrón salta sobre el perro perezoso.
    </p>
  }
</Example>

```html
<!-- [!code classes:capitalize] -->
<p class="capitalize">El rápido zorro marrón ...</p>
```

</Figure>

### Restableciendo la capitalización del texto

Usa la utilidad `normal-case` para preservar la capitalización original del texto de un elemento—típicamente usado para restablecer la capitalización en diferentes breakpoints:

<Figure>

<Example>
  {
    <p className="text-center text-lg font-medium text-gray-900 normal-case dark:text-gray-200">
      El rápido zorro marrón salta sobre el perro perezoso.
    </p>
  }
</Example>

```html
<!-- [!code classes:normal-case] -->
<p class="normal-case">El rápido zorro marrón ...</p>
```

</Figure>
