## Ejemplos

### Ejemplo básico

Usa utilidades como `text-blue-600` y `text-sky-400` para controlar el color del texto de un elemento:

<Figure>

<Example>
  {
    <div className="relative text-center text-xl leading-6 font-medium">
      <p className="text-blue-600 dark:text-sky-400">El rápido zorro marrón salta sobre el perro perezoso.</p>
    </div>
  }
</Example>

```html
<!-- [!code classes:text-blue-600,dark:text-sky-400] -->
<p class="text-blue-600 dark:text-sky-400">El rápido zorro marrón...</p>
```

</Figure>

### Cambiando la opacidad

Usa el modificador de opacidad de color para controlar la opacidad del color del texto de un elemento:

<Figure>

<Example>
  {
    <div className="space-y-4 text-center text-xl leading-6 font-medium">
      <p className="text-blue-600/100 dark:text-sky-400/100">El rápido zorro marrón salta sobre el perro perezoso.</p>
      <p className="text-blue-600/75 dark:text-sky-400/75">El rápido zorro marrón salta sobre el perro perezoso.</p>
      <p className="text-blue-600/50 dark:text-sky-400/50">El rápido zorro marrón salta sobre el perro perezoso.</p>
      <p className="text-blue-600/25 dark:text-sky-400/25">El rápido zorro marrón salta sobre el perro perezoso.</p>
    </div>
  }
</Example>

```html
<!-- [!code word:\/100] -->
<!-- [!code word:\/75] -->
<!-- [!code word:\/50] -->
<!-- [!code word:\/25] -->
<p class="text-blue-600/100 dark:text-sky-400/100">El rápido zorro marrón...</p>
<p class="text-blue-600/75 dark:text-sky-400/75">El rápido zorro marrón...</p>
<p class="text-blue-600/50 dark:text-sky-400/50">El rápido zorro marrón...</p>
<p class="text-blue-600/25 dark:text-sky-400/25">El rápido zorro marrón...</p>
```

</Figure>
