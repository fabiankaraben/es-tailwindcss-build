## Ejemplos

### Ejemplo básico

Usa utilidades como `decoration-sky-500` y `decoration-pink-500` para cambiar el color de la [decoración de texto](/docs/text-decoration-line) de un elemento:

<Figure>

<Example padding={false}>
  {
    <p className="mx-auto max-w-sm gap-4 p-8 text-sm/6 text-gray-900 dark:text-gray-200">
      Soy Derek, un astro-ingeniero con base en Tattooine. Me gusta construir X-Wings en{" "}
      <a href="#" className="font-bold underline decoration-sky-500 decoration-2 dark:text-gray-200">
        My Company, Inc
      </a>
      . Fuera del trabajo, me gusta{" "}
      <a href="#" className="font-bold underline decoration-pink-500 decoration-2 dark:text-gray-200">
        ver carreras de pods
      </a>{" "}
      y tener{" "}
      <a href="#" className="font-bold underline decoration-indigo-500 decoration-2 dark:text-gray-200">
        peleas de sables de luz
      </a>
      .
    </p>
  }
</Example>

```html
<!-- [!code classes:decoration-sky-500,decoration-pink-500,decoration-indigo-500] -->
<!-- prettier-ignore -->
<p>
  Soy Derek, un astro-ingeniero con base en Tattooine. Me gusta construir X-Wings
  en <a class="underline decoration-sky-500">My Company, Inc</a>. Fuera
  del trabajo, me gusta <a class="underline decoration-pink-500">ver carreras de pods</a>
  y tener <a class="underline decoration-indigo-500">peleas de sables de luz</a>.
</p>
```

</Figure>

### Cambiando la opacidad

Usa el modificador de opacidad de color para controlar la opacidad del color de la decoración de texto de un elemento:

<Figure>

<Example padding={false}>
  {
    <p className="mx-auto max-w-sm gap-4 p-8 text-sm/6 text-gray-900 dark:text-gray-200">
      Soy Derek, un astro-ingeniero con base en Tattooine. Me gusta construir X-Wings en{" "}
      <a href="#" className="font-bold underline decoration-sky-500/30 decoration-2 dark:text-gray-200">
        My Company, Inc
      </a>
      . Fuera del trabajo, me gusta{" "}
      <a href="#" className="font-bold underline decoration-pink-500/30 decoration-2 dark:text-gray-200">
        ver carreras de pods
      </a>{" "}
      y tener{" "}
      <a href="#" className="font-bold underline decoration-indigo-500/30 decoration-2 dark:text-gray-200">
        peleas de sables de luz
      </a>{" "}
      .
    </p>
  }
</Example>

```html
<!-- [!code word:\/30] -->
<!-- prettier-ignore -->
<p>
  Soy Derek, un astro-ingeniero con base en Tattooine. Me gusta construir X-Wings
  en <a class="underline decoration-sky-500/30">My Company, Inc</a>. Fuera
  del trabajo, me gusta <a class="underline decoration-pink-500/30">ver carreras de pods</a>
  y tener <a class="underline decoration-indigo-500/30">peleas de sables de luz</a>.
</p>
```

</Figure>

### Aplicando en hover

<TargetingSpecificStates property="text-decoration-color">

<Figure hint="Pasa el cursor sobre el texto para ver el comportamiento esperado">

<Example>
  {
    <div className="text-center text-lg font-medium text-gray-900 no-underline dark:text-gray-200">
      El{" "}
      <a
        href="https://en.wikipedia.org/wiki/The_quick_brown_fox_jumps_over_the_lazy_dog"
        target="blank"
        className="underline hover:decoration-pink-500"
      >
        rápido zorro marrón
      </a>{" "}
      salta sobre el perro perezoso.
    </div>
  }
</Example>

```html
<!-- [!code classes:hover:decoration-pink-500] -->
<p>El <a href="..." class="underline hover:decoration-pink-500 ...">rápido zorro marrón</a> salta sobre el perro perezoso.</p>
```

</Figure>

</TargetingSpecificStates>
