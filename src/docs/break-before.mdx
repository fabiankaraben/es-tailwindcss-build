## Ejemplos

### Ejemplo básico

Usa utilidades como `break-before-column` y `break-before-page` para controlar cómo un salto de columna o página debe comportarse antes de un elemento:

```html
<!-- [!code classes:break-before-column] -->
<div class="columns-2">
  <p>Bueno, déjame decirte algo, ...</p>
  <p class="break-before-column">Claro, adelante, ríete...</p>
  <p>Tal vez podamos vivir sin...</p>
  <p>Mira. Si crees que esto es...</p>
</div>
```
