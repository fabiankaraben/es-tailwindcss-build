## Ejemplos

### Alineando texto a la izquierda

Usa la utilidad `text-left` para alinear el texto de un elemento a la izquierda:

<Figure>

<Example padding={false}>
  {
    <div className="px-4 sm:px-0">
      <div className="mx-auto max-w-lg p-8 text-sm/6 text-gray-900 dark:text-gray-200">
        <p className="text-left">
          Así que empecé a caminar hacia el agua. No les mentiré, muchachos, estaba aterrorizado. Pero seguí adelante, y mientras pasaba las rompientes una extraña calma me invadió. No sé si fue intervención divina o la afinidad de todos los seres vivos, pero te digo Jerry, en ese momento, I <em>was</em> a marine biologist.
        </p>
      </div>
    </div>
  }
</Example>

```html
<!-- [!code classes:text-left] -->
<p class="text-left">Así que empecé a caminar hacia el agua...</p>
```

</Figure>
