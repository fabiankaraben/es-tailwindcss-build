## Ejemplos

### Ejemplo básico

Usa utilidades `indent-<number>` como `indent-2` e `indent-8` para establecer la cantidad de espacio vacío (sangría) que se muestra antes del texto en un bloque:

<Figure>

<Example>
  {
    <p className="mx-auto max-w-sm indent-8 text-gray-900 dark:text-gray-200">
      Así que empecé a caminar hacia el agua. No les mentiré chicos, estaba aterrado. Pero seguí adelante, y mientras pasaba las olas rompientes una extraña calma me invadió. No sé si fue intervención divina o la hermandad de todos los seres vivos, pero les digo Jerry en ese momento, <em>era</em> un biólogo marino.
    </p>
  }
</Example>

```html
<!-- [!code classes:indent-8] -->
<p class="indent-8">Así que empecé a caminar hacia el agua...</p>
```

</Figure>

### Usando valores negativos

Para usar un valor de sangría de texto negativo, prefija el nombre de la clase con un guion para convertirlo en un valor negativo:

<Figure>

<Example>
  {
    <p className="mx-auto max-w-sm -indent-8 text-gray-900 dark:text-gray-200">
      Así que empecé a caminar hacia el agua. No les mentiré chicos, estaba aterrado. Pero seguí adelante, y mientras pasaba las olas rompientes una extraña calma me invadió. No sé si fue intervención divina o la hermandad de todos los seres vivos, pero les digo Jerry en ese momento, <em>era</em> un biólogo marino.
    </p>
  }
</Example>

```html
<!-- [!code classes:-indent-8] -->
<p class="-indent-8">Así que empecé a caminar hacia el agua...</p>
```

</Figure>

### Usando un valor personalizado

<UsingACustomValue element="p" utility="indent" name="text indentation" value="50%" variable="indentation" />
