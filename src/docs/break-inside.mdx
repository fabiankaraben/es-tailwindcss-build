## Ejemplos

### Ejemplo básico

Usa utilidades como `break-inside-column` y `break-inside-avoid-page` para controlar cómo un salto de columna o página debe comportarse dentro de un elemento:

```html
<!-- [!code classes:break-inside-avoid-column] -->
<div class="columns-2">
  <p>Bueno, déjame decirte algo, ...</p>
  <p class="break-inside-avoid-column">Claro, adelante, ríete...</p>
  <p>Tal vez podamos vivir sin...</p>
  <p>Mira. Si crees que esto es...</p>
</div>
```
