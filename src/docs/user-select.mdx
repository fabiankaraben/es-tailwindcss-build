## Ejemplos

### Deshabilitando la selección de texto

Usa la utilidad `select-none` para evitar la selección de texto en un elemento y sus hijos:

<Figure hint="Intenta seleccionar el texto para ver el comportamiento esperado">

<Example>
  {
    <div className="flex justify-center">
      <div className="dark:highlight-white/5 inline-flex rounded-lg bg-white px-4 py-3 text-center font-sans text-sm font-semibold text-gray-900 ring-1 ring-gray-900/5 select-none dark:bg-gray-800 dark:text-gray-200 dark:ring-0">
        El rápido zorro marrón salta sobre el perro perezoso.
      </div>
    </div>
  }
</Example>

```html
<!-- [!code classes:select-none] -->
<div class="select-none ...">El rápido zorro marrón salta sobre el perro perezoso.</div>
```

</Figure>
