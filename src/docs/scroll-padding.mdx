### Usar valores negativos

Para usar un valor negativo de scroll padding, prefija el nombre de la clase con un guion para convertirlo en un valor negativo:

```html
<!-- [!code classes:-scroll-ps-6] -->
<div class="-scroll-ps-6 snap-x ...">
  <!-- ... -->
</div>
```
