### Usando el contenedor de breakpoints

Usa la utilidad `container` para establecer el ancho máximo de un elemento para que coincida con el `min-width` del breakpoint actual. Esto es útil si prefieres diseñar para un conjunto fijo de tamaños de pantalla en lugar de intentar acomodar un viewport completamente fluido:

```html
<!-- [!code classes:container] -->
<div class="container">
  <!-- ... -->
</div>
```

Ten en cuenta que, a diferencia de los contenedores que puedes haber usado en otros frameworks, el contenedor de Tailwind no se centra automáticamente y no tiene ningún padding horizontal incorporado. Usa `mx-auto` y las utilidades `px-<number>` para agregarlos:

```html
<!-- [!code classes:mx-auto,px-4] -->
<div class="container mx-auto px-4">
  <!-- ... -->
</div>
```
