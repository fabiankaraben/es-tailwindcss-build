### Ejemplo básico

Usa la utilidad `mask-repeat` para repetir la imagen de máscara tanto vertical como horizontalmente:

      </Stripes>
    </div>
  }
</Example>

{/* prettier-ignore */}
```html
<!-- [!code classes:mask-repeat] -->

```

</Figure>
