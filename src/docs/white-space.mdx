## Ejemplos

### Normal

Usa la utilidad `whitespace-normal` para hacer que el texto se ajuste normally dentro de un elemento. Las líneas nuevas y los espacios se colapsarán:

<Figure>

<Example padding={false}>
  {
    <div className="mx-auto max-w-sm border-x border-x-pink-400/30 py-8 text-gray-900 dark:text-gray-200">
      <p
        className="whitespace-normal"
        children={
          "¡Hola a todos!\n\nYa casi es 2022       y todavía no sabemos si hay             extraterrestres viviendo entre nosotros, ¿o sí? Tal vez la persona que escribe esto sea un extraterrestre.\n\nNunca lo sabrás."
        }
      />
    </div>
  }
</Example>

```html
<!-- [!code classes:whitespace-normal] -->
<!-- prettier-ignore -->
<p class="whitespace-normal">¡Hola a todos!

Ya casi es 2022       y todavía no sabemos si hay             extraterrestres viviendo entre nosotros, ¿o sí? Tal vez la persona que escribe esto sea un extraterrestre.

Nunca lo sabrás.</p>
```

</Figure>

      />
    </div>

</Example>

```html
<!-- [!code classes:whitespace-nowrap] -->

Nunca lo sabrás.</p>
```

</Figure>

### Romper espacios (Break Spaces)

Usa la utilidad `whitespace-break-spaces` para preservar las líneas nuevas y los espacios dentro de un elemento. El espacio en blanco al final de las líneas no colgará, sino que se ajustará a la siguiente línea:

<Figure>

<Example padding={false}>
  {
    <div className="mx-auto max-w-sm border-x border-x-pink-400/30 py-8 text-gray-900 dark:text-gray-200">
      <p
        className="whitespace-break-spaces"
        children={
          "¡Hola a todos!\n\nYa casi es 2022       y todavía no sabemos si hay             extraterrestres viviendo entre nosotros, ¿o sí? Tal vez la persona que escribe esto sea un extraterrestre.\n\nNunca lo sabrás."
        }
      />
    </div>
  }
</Example>

```html
<!-- [!code classes:whitespace-break-spaces] -->
<!-- prettier-ignore -->
<p class="whitespace-break-spaces">¡Hola a todos!

Ya casi es 2022       y todavía no sabemos si hay             extraterrestres viviendo entre nosotros, ¿o sí? Tal vez la persona que escribe esto sea un extraterrestre.

Nunca lo sabrás.</p>
```

</Figure>
