## Ejemplos

### Usar desplazamiento suave

Usa la utilidad `scroll-smooth` para habilitar el desplazamiento suave dentro de un elemento:

```html
<!-- [!code classes:scroll-smooth] -->
<html class="scroll-smooth">
  <!-- ... -->
</html>
```

Establecer el `scroll-behavior` solo afecta a los eventos de scroll que son activados por el navegador.

### Usar desplazamiento normal

Usa la utilidad `scroll-auto` para revertir al comportamiento predeterminado del navegador para el desplazamiento:

```html
<!-- [!code classes:scroll-auto] -->
<html class="scroll-smooth md:scroll-auto">
  <!-- ... -->
</html>
```
