## Ejemplos

### Antialiasing en escala de grises

Utiliza la utilidad `antialiased` para renderizar texto usando antialiasing en escala de grises:

<Figure>

<Example>
  {
    <p className="text-center text-lg font-medium text-gray-900 antialiased dark:text-gray-200">
      El rápido zorro marrón salta sobre el perro perezoso.
    </p>
  }
</Example>

```html
<!-- [!code classes:antialiased] -->
<p class="antialiased ...">El rápido zorro marrón ...</p>
```

</Figure>

### Antialiasing subpíxel

Utiliza la utilidad `subpixel-antialiased` para renderizar texto usando antialiasing subpíxel:

<Figure>

<Example>
  {
    <p className="text-center text-lg font-medium text-gray-900 subpixel-antialiased dark:text-gray-200">
      El rápido zorro marrón salta sobre el perro perezoso.
    </p>
  }
</Example>

```html
<!-- [!code classes:subpixel-antialiased] -->
<p class="subpixel-antialiased ...">El rápido zorro marrón ...</p>
```

</Figure>
