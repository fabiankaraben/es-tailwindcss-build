### Usando valores negativos

Para usar un valor negativo de top/right/bottom/left, antepone un guion al nombre de la clase para convertirlo en un valor negativo:

<Figure>

<Example>
  {
    <div className="flex justify-center font-mono text-sm leading-6 font-bold text-white">
      <div className="relative size-18 rounded-lg sm:h-32 sm:w-32">
        <div className="absolute inset-0">
          <Stripes border className="h-full rounded-lg" />
        </div>
        <div className="absolute -top-4 -left-4 flex items-center justify-center rounded-lg bg-purple-500 p-4"></div>
      </div>
    </div>
  }
</Example>

```html
<!-- [!code classes:-left-4,-top-4] -->
<div class="relative size-32 ...">
  <div class="absolute -top-4 -left-4 size-14 ..."></div>
</div>
```

</Figure>

### Usando propiedades lógicas

Usa utilidades `start-<number>` o `end-<number>` como `start-0` y `end-4` para establecer las [propiedades lógicas](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_Logical_Properties/Basic_concepts) `inset-inline-start` y `inset-inline-end`, que se asignan al lado izquierdo o derecho según la dirección del texto:

<Figure>

<Example>
  {
    <div className="grid grid-cols-2 place-items-center gap-x-4">
      <div className="flex flex-col items-start gap-y-4">
        <p className="text-sm font-medium">De izquierda a derecha</p>
        <div className="relative size-18 rounded-lg sm:size-32">
          <div className="absolute inset-0">
            <Stripes border className="h-full rounded-lg" />
          </div>
          <div className="absolute top-0 left-0 flex size-14 items-center justify-center rounded-lg bg-purple-500 p-4"></div>
        </div>
      </div>
      <div className="flex flex-col items-end gap-y-4">
        <p className="text-sm font-medium">De derecha a izquierda</p>
        <div className="relative size-18 rounded-lg sm:size-32">
          <div className="absolute inset-0">
            <Stripes border className="h-full rounded-lg" />
          </div>
          <div className="absolute top-0 right-0 flex size-14 items-center justify-center rounded-lg bg-purple-500 p-4"></div>
        </div>
      </div>
    </div>
  }
</Example>

```html
<!-- [!code classes:start-0] -->
<div dir="ltr">
  <div class="relative size-32 ...">
    <div class="absolute start-0 top-0 size-14 ..."></div>
  </div>
  <div>
    <div dir="rtl">
      <div class="relative size-32 ...">
        <div class="absolute start-0 top-0 size-14 ..."></div>
      </div>
      <div></div>
    </div>
  </div>
</div>
```

</Figure>

Para mayor control, también puedes usar los [modificadores LTR y RTL](/docs/hover-focus-and-other-states#rtl-support) para aplicar estilos específicos condicionalmente según la dirección actual del texto.
