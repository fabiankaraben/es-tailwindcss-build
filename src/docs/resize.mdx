## Ejemplos

### Redimensionar en todas las direcciones

Usa `resize` para hacer que un elemento sea redimensionable horizontal y verticalmente:

<Figure hint="Arrastra el controlador del textarea en la demostración para ver el comportamiento esperado">

<Example>
  {
    <textarea
      rows="2"

    ></textarea>
  }
</Example>

```html
<!-- [!code classes:resize] -->
<textarea class="resize rounded-md ..."></textarea>
```

</Figure>
