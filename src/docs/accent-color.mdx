## Ejemplos

### Estableciendo el color de acento

Usa utilidades como `accent-rose-500` y `accent-lime-600` para cambiar el color de acento de un elemento:

<Figure>

<Example>
  {
    <div className="flex flex-wrap justify-center gap-6">
      <label className="flex items-center space-x-2">
        <input type="checkbox" defaultChecked />
        <div className="text-sm font-semibold text-gray-900 dark:text-gray-200">Predeterminado del navegador</div>
      </label>
      <label className="flex items-center space-x-2">
        <input type="checkbox" className="accent-pink-500" defaultChecked />
        <div className="text-sm font-semibold text-gray-900 dark:text-gray-200">Personalizado</div>
      </label>
    </div>
  }
</Example>

```html
<!-- [!code classes:accent-pink-500] -->
<label>
  <input type="checkbox" checked />
  Predeterminado del navegador
</label>
<label>
  <input class="accent-pink-500" type="checkbox" checked />
  Personalizado
</label>
```

</Figure>

Esto es útil para estilizar elementos como checkboxes y grupos de radio sobrescribiendo el color predeterminado del navegador.

### Cambiando la opacidad

Usa el modificador de opacidad de color para controlar la opacidad del color de acento de un elemento:

<Figure>

<Example>
  {
    <div className="flex flex-wrap justify-center gap-6">
      <label className="flex items-center space-x-2">
        <input type="checkbox" defaultChecked className="accent-purple-500/25" />
        <div className="text-sm font-semibold text-gray-900 dark:text-gray-200">accent-purple-500/25</div>
      </label>
      <label className="flex items-center space-x-2">
        <input type="checkbox" className="accent-purple-500/75" defaultChecked />
        <div className="text-sm font-semibold text-gray-900 dark:text-gray-200">accent-purple-500/75</div>
      </label>
    </div>
  }
</Example>

```html
<!-- [!code word:/25] -->
<!-- [!code word:/75] -->
<input class="accent-purple-500/25" type="checkbox" checked />
<input class="accent-purple-500/75" type="checkbox" checked />
```

</Figure>

Establecer la opacidad del color de acento tiene soporte limitado en navegadores y actualmente solo funciona en Firefox.

### Usando un valor personalizado

<UsingACustomValue
  element="input"
  elementAttributes={{ type: "checkbox" }}
  utility="accent"
  value="#50d71e"
  name="accent color"
  variable="accent-color"
/>
