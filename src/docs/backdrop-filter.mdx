### Eliminando filtros

Usa la utilidad `backdrop-filter-none` para eliminar todos los filtros de fondo aplicados a un elemento:

```html
<!-- [!code classes:md:backdrop-filter-none] -->
<div class="backdrop-blur-md backdrop-brightness-150 md:backdrop-filter-none"></div>
```
