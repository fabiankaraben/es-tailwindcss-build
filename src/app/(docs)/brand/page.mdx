## Acuerdo de uso de marca registrada

El nombre y los logotipos de Tailwind son marcas registradas de Tailwind Labs Inc.

No puedes usar el nombre o los logotipos de Tailwind de ninguna manera que pueda implicar erróneamente una conexión oficial o un respaldo de Tailwind Labs Inc. Cualquier uso del nombre o logotipos de Tailwind que pueda causar confusión a los clientes no está permitido.

Esto incluye nombrar un producto o servicio de una manera que enfatice la marca Tailwind, como "Tailwind Themes" o "Tailwind Studio", así como en nombres de dominio como "tailwindkits.com".

En cambio, debes usar tu propia marca de una manera que la distinga claramente de Tailwind CSS. Ejemplos que están permitidos incluyen "ComponentStudio para Tailwind CSS" o "UtilityThemes: Plantillas para Tailwind CSS".

Además, no puedes usar nuestras marcas registradas para camisetas, pegatinas u otra mercancía sin consentimiento escrito explícito.
